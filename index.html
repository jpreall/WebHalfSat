HalfSat

Modeling Sequencing Saturation in 10X Genomics data

Load a local 10x Genomics QC summary file (`web_summary.html` or `metrics_summary.json`) and interactively predict read depth, UMIs per cell, and sequencing saturation. This tool can be useful for planning additional sequencing or evaluating the efficiency of a run.

How The Model Works (click to expand)

The app fits simple saturating curves to your downsampling points. It uses the fitted parameters to predict values at new read depths and to estimate the reads/cell needed to hit a target.

Two related saturation models are available for the sequencing saturation curve. A Lander-Waterman-like fit is more mechanistically tied to library complexity / duplicate discovery, while a Michaelis-Menten-like (hyperbolic) fit is a simpler approximation that is often very stable and works well across the observed downsampling range.

In this app, the model selector only affects the sequencing saturation plot. The UMIs/genes curve remains a separate Michaelis-Menten-like saturating fit because UMIs/genes-vs-reads are typically better treated as empirical saturation curves than strict Lander-Waterman coverage models.

Sequencing Saturation

M-M-like: s(R) = R / (R + a)

M-M-like inverse: R = (a · s) / (1 - s)

L-W-like: s(R) = 1 - [(1 - e^(-R/X)) · X / R]

L-W-like target: solve s(R) = target for R (numerically)

a is the half-saturation parameter (reads/cell at which the curve reaches half of its maximum). The maximum saturation is fixed at 1.0.

If you choose the Lander-Waterman-like model in the selector, the left plot uses an alternative saturating form based on a library-complexity-style model. The displayed “Sequencing HalfSat Point” is still reported as the reads/cell value where the fitted saturation reaches 50%.

In the L-W-like form above, X is the fitted library-complexity scale parameter. Because the inverse does not have a simple closed form here, the app solves for target reads numerically.

UMIs/Genes vs Reads

M(R) = (b · R) / (R + a)

R = (a · M) / (b - M)

a is the half-saturation parameter and b is the fitted maximum (asymptote). The lower plot uses UMIs when available, and falls back to genes for some web summary files.

This lower curve keeps its own independent HalfSat point based on the UMIs/genes fit. Changing the sequencing saturation model does not change the UMIs/genes fit or its HalfSat marker.
