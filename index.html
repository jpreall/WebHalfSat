Halfsat

Modeling Sequencing Saturation in 10X Genomics data

Load a local 10x file (`web_summary.html` or `metrics_summary.json`) and interactively predict read depth, UMIs per cell, and sequencing saturation.

How The Model Works (click to expand)

The app fits simple saturating curves to your downsampling points. It uses the fitted parameters to predict values at new read depths and to estimate the reads/cell needed to hit a target.

Sequencing Saturation

s(R) = R / (R + a)

R = (a · s) / (1 - s)

a is the half-saturation parameter (reads/cell at which the curve reaches half of its maximum). The maximum saturation is fixed at 1.0.

UMIs/Genes vs Reads

M(R) = (b · R) / (R + a)

R = (a · M) / (b - M)

a is the half-saturation parameter and b is the fitted maximum (asymptote). The lower plot uses UMIs when available, and falls back to genes for some web summary files.
